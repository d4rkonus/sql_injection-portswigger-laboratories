--------

Una inyección SQL o **SQL Injection** es una de las vulnerabilidades web más comunes y letales que existen. Ocurren cuando un atacante inyecta código SQL malicioso en una consulta o *query* para que después el servicio o aplicación web la envíe a la base de datos donde se almacena la información. 

De esta forma, el atacante puede obtener de forma ilegal o no autorizada los datos que se almacenan en la base de datos; pueden ir desde unos datos específicos hasta toda la base de datos en su conjunto.

Una mala práctica en la seguridad de los datos en grandes bases de datos puede llevar a que con una simple vulnerabilidad web puedan desatarse consecuencias muy dañinas para la información de los usuarios.
